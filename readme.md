### Installing
**Arch:** [jfetch-git](https://aur.archlinux.org/packages/jfetch-git/) (AUR)  
**Other Distros:** clone the repo and run `sudo make install`

### Customization
`jfetch` can be configured and customized modifying the source, for example:
```
terminal = ${wm}
```
